# StoryTale


## 项目概述
- **技术栈**: Flutter, TTS, Deepseek Chater
- **主要功能**: 
  - 根据输入单词自动生成短文，并提供 TTS 朗读
  - 支持文章模式 / 句子模式 / 随身听模式
- **项目状态**: 即将上线！
